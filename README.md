# Playwright Automation Test

Automation test project using Playwright for CURA Healthcare Service.

## Tech Stack

- Playwright
- JavaScript
- Node.js
- GitHub Actions

## Installation

Clone repository:

git clone <repository-url>

Install dependencies:

npm install

Install browser:

npx playwright install

## Run Test

Run all tests:

npm test

Run specific test:

npx playwright test tests/login.spec.js

Generate report:

npx playwright show-report
